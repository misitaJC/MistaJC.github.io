Random Color Grid

The Interactive Color Grid is a web application that allows users to engage with a grid of color buttons. Each button, when clicked, randomly changes its background and text to a new color from a predefined array. Styled with CSS for a responsive layout and powered by JavaScript, this application offers a dynamic way to explore colors. It's not only a fun tool for color enthusiasts but also a practical example of basic web development techniques, showcasing the use of arrays, loops, and event listeners in action.

2048-Master

This web adaptation of the "2048" game is designed with HTML for structure, CSS for its sleek look, and JavaScript to handle the game's logic and user interactions. Players aim to merge tiles on a 4x4 grid to create the tile numbered 2048. The game's objective is achieved by combining like numbers through swipes, with a clean interface, smooth animations, and a scoring system that reflects each successful merge. Its simple yet challenging gameplay ensures hours of entertainment and mental challenge.

Lucky-Draw

This web project is an interactive random prize draw game that simulates a spinning wheel with various prize sections. Built with HTML for structure, CSS for styling, and JavaScript for functionality, the game allows users to click a button to spin the wheel and randomly determine a prize. It features a visually appealing interface and dynamic interactions, providing an engaging online gaming experience, and is a fun way to bring the excitement of a physical prize wheel to a digital platform.

Jigsaw-Puzzle

The project is an interactive 3x3 sliding puzzle game designed for web platforms. It features a clean interface with a timer to monitor the time taken to solve the puzzle and controls to pause or reset the game. The game is styled with CSS for aesthetic presentation and uses JavaScript for its core functionality, including tile movement logic and user interaction. This sliding puzzle game is a fun way to challenge cognitive skills with a simple yet engaging gameplay experience.
